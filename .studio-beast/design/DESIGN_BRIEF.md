# Studio Beast Design Brief: PermitPath UX/UI Overhaul

## Project Overview
**Mission:** Transform the PermitPath contractor permit assistant app with a professional, trustworthy design system.

## Design System Foundation
- **Philosophy:** Trust, Clarity, Speed
- **Primary:** Blueprint Blue (#0A3A7E)
- **Background:** Parchment White (#F7FAFC)
- **Accent:** Safety Orange (#DD6B20)
- **Success:** Forest Green (#2F855A)
- **Error:** Crimson Red (#C53030)
- **Text:** Charcoal (#2D3748), Steel Gray (#4A5568)

## Screens to Redevelop (6 total)
1. Home Page - Hero, CTAs, feature highlights
2. New Job / Wizard Flow - Job selection, form steps
3. AI Permit Assistant - Chat interface, photo upload
4. My Jobs Page - Card list, empty state, status badges
5. Help & Support Page - Contact options, FAQ accordions
6. Settings Page - Preferences, account management

## Image Assets Needed
- Hero background (architect + blueprint, 16:9)
- Job type icons: Roofing, Water Heater, Electrical Panel, AC/HVAC (SVG style)
- AI Assistant banner (neural network + blueprint, 3:1)
- Empty state illustration (clipboard, 4:3)
- Help icons: Chat AI, Call County, Email Support

## Non-Negotiables
- WCAG 2.1 Level AA compliance
- 15+ edge cases
- 12+ e2e test scenarios
- 10+ acceptance criteria
- Clean git commits throughout
