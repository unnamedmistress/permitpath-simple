# communication/

This folder is the single source of truth for **durable artifacts** produced by the agent system.

Rules:
- All plans, routing decisions, handoffs, and written comms must be saved here.
- Artifacts must be committed and pushed to GitHub.
- At the end of each run, the agent must provide GitHub links to any files created/updated here.

Naming:
- YYYY-MM-DD_HHMMZ_<slug>.md
